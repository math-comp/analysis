    # Negligibility NB: See CONTRIBUTING.md for an introduction to HB concepts and commands. ``` mu.-negligible A == A is mu negligible measure_is_complete mu == the measure mu is complete {ae mu, P} == P holds almost everywhere for the measure mu, declared as an instance of the type of filters P must be of the form forall x, Q x. Prefer this notation when P is an existing statement (i.e., a definition) that needs to be relativised. The notation used the definition `almost_everywhere`. \forall x \ae mu, P x == equivalent to {ae mu, forall x, P x} Prefer this notation when the statement forall x, P x does not stand alone. f = g %[ae mu in D ] == f is equal to g almost everywhere in D f = g %[ae mu] == f is equal to g almost everywhere mu-.null_set == (measure-theoretic) null sets m1 `<< m2 == m1 is absolutely continuous w.r.t. m2 or m2 dominates m1 content_dominates mu nu == forall A, measurable A -> mu A = 0 -> nu A = 0 ```
