    # Convexity This file provides a small account of convexity using convex spaces, to be completed with material from InfoTheo. ``` convex_quasi_associative == quasi-associativity of the operator of convex spaces isConvexSpace R T == interface for convex spaces with R : numDomainType The HB class is ConvexSpace. a <| t |> b == convexity operator ``` For `R : numDomainType`, `E : lmodType R` and `R` itself are shown to be convex spaces with the following aliases: ``` convex_lmodType E == E : lmodType R as a convex space convex_numDomainType R == R : numDomainType as a convex space ``` ``` convex_set A == A : set M is a convex set M is an Lmodule over R : numDomainType convex_function f == f : E -> R is a convex function E is an Lmodule over R : numFieldType ```
