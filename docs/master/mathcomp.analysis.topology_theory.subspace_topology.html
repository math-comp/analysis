Its use is however likely to be later superseded by a better (compositional) mechanism. {within A, continuous f} := continuous (from_subspace A f)) subspace_ent A == subspace entourages subspace_ball A == balls of the pseudometric subspace structure continuousSubspaceType A B == type of continuous functions from set A to set B with domain subspace A The HB structure is ContinuousFun. ```
