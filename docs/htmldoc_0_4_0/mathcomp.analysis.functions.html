Theory of functions
This file provides a theory of functions whose domain and codomain are represented by sets.
set_fun A B f == f : aT -> rT is a function with domain A : set aT and codomain B : set rT set_surj A B f == f is surjective set inj A B f == f is injective set_bij A B f == f is bijective {fun A >-> B } == type of functions f : aT -> rT from A : set aT to B : set rT {oinv aT >-> rT} == type of functions with a partial inverse {oinvfun A >-> B} == combination of {fun A >-> B} and {oinv aT >-> rT} {inv aT >-> rT} == type of functions with an inverse f ^-1 == inverse of f : {inv aT >-> rT} {invfun A >-> B} == combination of {fun A >-> B} and {inv aT >-> rT} {surjfun A >-> B} == type of surjective functions {injfun A >-> B} == type of injective functions {bij A >-> B} == combination of {injfun A >-> B} and {surjfun A >-> B}
Section function_space == canonical ringType and lmodType structures for functions whose range is a ringType, comRingType,or lmodType. fctE == multi-rule for fct
